# Haversine Distance Calculator & Profiler
This project includes two programs: ```generate_json``` and ```haversine```.<br>
 - ```generate_json```: Generates a .json file containing N number of pairs. One pair consists of two points. One point consists of two coordinates (X and Y), each represented by a double-precision floating-point number (8 bytes).<br>
 - ```haversine```: Processes the .json file generated by generate_json and calculates the Haversine distance between the two points of each pair. The program then computes and outputs the average of all the calculated Haversine distances.<br>
Both programs can be compiled in Normal mode (profiler off), Profiler mode (profiler on, using custom instrumentation), and Debug mode. (For more information, see the Compilation section).<br>
## Context
This project is a solution for one of the homework assignments in Casey Muratori's Performance-Aware Programming course.<br>

Note: In the file src/generator/generator.c, there are a few lines of code copy-pasted from the official computer_enhance GitHub repo to ensure mathematical consistency with the course.<br>
## Compilation
### Build with Makefile
Normal build:
```sh
make all
```
Build with profiler on:
```sh
make prof
```
Build with debug flags:
```sh
make debug
```

### Compile from command line
This project is structured as a Unity Build (Jumbo Build), meaning you only need to compile a single .c file for each program.<br>
Path to the ```generate_json``` source: `src/generator/generator_main.c`<br>
Path to the ```haversine``` source: `src/main_haversine.c`<br>
```sh
gcc <source_file.c> -lm
```
*(For more addition flags, refer to the Makefile)*

## Example
1. Generating Data<br>
*(Example input)*<br>
```sh
./generate_json_prof cluster 14234 10000000
```
*(Example output)*<br>
```sh
Method: cluster
Seed: 14234
Pair count: 10000000
Expected sum: 1765.1954259474553055
```
2. Running Haversine (Normal Mode)<br>
